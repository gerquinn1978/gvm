**Evidence (Muller-Brockmann):** Authority — originating theorist of grid-based design. Adoption — lineage to CSS Grid, Bootstrap. Work Influence — foundational reference.

**Evidence (Bringhurst):** Authority — "the typographer's bible." Adoption — 400,000+ copies; modular scale at Apple, Google, IBM. Work Depth — exhaustive typography treatment.

**Activation signals:** Brochure spec, designed output, any deliverable with specific layout requirements

**Key techniques to apply:**

- **Grid system** (Muller-Brockmann): All layout on a modular grid. Content aligns to grid lines. The grid creates visual order and proportional relationships.
- **Typographic scale** (Bringhurst): Font sizes follow a harmonious ratio (e.g., 1.2 or 1.25 scale). Body text → subheadings → headings follow the scale consistently.
- **Hierarchy through type** (Bringhurst, Ch. 2): Use weight, size, and style to create clear hierarchy. Avoid relying on colour alone.

**When specifying designed output:**
- Reference the shared design system's grid and typography patterns
- Specify content blocks and their grid placement
- Define the typographic hierarchy for the specific document
